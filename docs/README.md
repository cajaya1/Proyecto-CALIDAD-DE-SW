# Diagramas del Proyecto

Este directorio contiene diagramas en formato Mermaid para visualizar la arquitectura, UML y el esquema de base de datos.

## Cómo verlos
- En VS Code, instala la extensión "Markdown Preview Mermaid Support" o similar.
- Abre cada archivo `.md` y utiliza la vista previa.
- En GitHub, los bloques Mermaid se renderizan automáticamente.

## Archivos
- `diagrams/architecture.md`: Arquitectura general Frontend ↔ Backend ↔ DB.
- `diagrams/uml-classes.md`: UML de clases simplificado para controladores y modelos.
- `diagrams/uml-sequence-chatbot.md`: Secuencia del flujo de mensaje del Chatbot.
- `diagrams/uml-sequence-reservation.md`: Secuencia del ciclo de vida de Reservas.
- `diagrams/db-schema.md`: Diagrama ER del esquema de base de datos.
